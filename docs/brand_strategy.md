# EVision — Brand Strategy v1

---

## 1. Brand Core

### Миссия

> Превратить глобальные экономические данные в интеллект, доступный каждому, кто принимает решения на основе фактов.

### Vision

EVision — стандарт экономической грамотности нового поколения. Не инструмент. Точка зрения.

### Ценности

| Ценность | Суть |
|---|---|
| **Точность** | Данные — это ответственность. Ни одного числа без источника. |
| **Ясность** | Сложное становится понятным. Визуализация — не украшение, а аргумент. |
| **Суверенность** | Пользователь сам формирует вывод. Нет пресетного мышления. |
| **Масштаб** | Мыслить страновым уровнем. Сравнивать несопоставимое. |
| **Интеллект** | AI — не магия, а усилитель аналитики. Всегда объяснимый. |

### Brand Personality

```
Архетип: Мудрец + Правитель
──────────────────────────────────────────────────────────
Мудрец:     знает больше, чем показывает. Не кричит — аргументирует.
Правитель:  уверен, структурирован, держит рамку. Задаёт стандарт.

НЕ является: стартап-дружелюбным, игровым, тревожным, кричащим.
```

### Tone of Voice

| Канал | Тон | Пример |
|---|---|---|
| UI-лейблы | Лаконичный, нейтральный | «Коэффициент Джини» / «Gini Index» |
| Пустые состояния | Уверенный, не извиняющийся | «Выберите страну для начала анализа» |
| Ошибки | Прямой, информативный | «Данные недоступны. Источник: World Bank» |
| Онбординг | Интеллектуальный, краткий | «EVision анализирует. Вы решаете.» |
| AI-объяснения | Аналитический, взвешенный | «Тренд умеренно восходящий. Разрыв со средним по региону — 12 п.п.» |

## 2. Три бренд-концепции

---

### Концепция A — «Signal»

**Идея**
Экономика — это поток сигналов. EVision фильтрует шум и выделяет значимое. Образ: радарный экран, где страны — точки, индикаторы — волны. Пользователь — оператор, который читает мир.

**Настроение**
Холодный интеллект. Ночной контрольный центр. Уверенность без эмоций.

**Визуальные ассоциации**
- Цвет: глубокий navy `#0B1628`, electric teal `#00C9B1`, белый текст
- Графика: радиальные сетки, пульсирующие точки на карте, тонкие линии сигналов
- Типографика: моноширинные акценты для данных, geometric sans для текста
- Анимации: fade-in волнами, плавный trace линий графиков
- Плотность: высокая, информационно насыщенная

**Слоган**

| Язык | Слоган |
|---|---|
| RU | «Читай сигналы. Понимай мир.» |
| EN | «Cut through the noise. Read the world.» |
| KZ | «Сигналды оқы. Дүниені тан.» |

**Ключевые слова:** signal, radar, frequency, clarity, precision, operator

**Risk notes**
- Может восприниматься как «для военных/разведки» — осторожно в KZ-аудитории
- Радарная эстетика требует высокого UI-качества исполнения, иначе выглядит как аматорский дашборд
- Сложен в адаптации для светлой темы

---

### Концепция B — «Meridian»

**Идея**
Меридианы делят мир на координаты. EVision даёт систему координат для экономики. Образ: глобус, сечения, пересечения линий — каждая страна занимает своё место в пространстве данных. Сравнение — это позиционирование.

**Настроение**
Картографическая строгость. Атлас для аналитика. Академическая точность с modern-finish.

**Визуальные ассоциации**
- Цвет: глубокий slate `#1E2A3A`, золотой акцент `#C9A84C`, off-white `#F4F1EB`
- Графика: тонкие сетки координат, контурные карты, минималистичные scatter-plots
- Типографика: serif для заголовков (интеллект, авторитет), sans для данных
- Анимации: zoom-in на регион, плавное перемещение между координатами
- Плотность: средняя, академический white space

**Слоган**

| Язык | Слоган |
|---|---|
| RU | «Каждая экономика — на своих координатах.» |
| EN | «Position your understanding. Map every economy.» |
| KZ | «Әрбір экономика — өз координатасында.» |

**Ключевые слова:** coordinates, position, atlas, meridian, cartography, place

**Risk notes**
- «Meridian» — существующее название у ряда fintech-компаний, нужна проверка товарного знака
- Serif + золото рискует уйти в «банковский бренд 2010-х» без сильного modern balance
- Академический тон может восприниматься как «скучный» для студенческой аудитории

---

### Концепция C — «Apex View»

**Идея**
Видеть сверху. Apex — вершина пирамиды данных. EVision даёт стратегический обзор, который недоступен тем, кто смотрит снизу. Образ: панорама с высоты, где отдельные числа складываются в архитектуру экономики.

**Настроение**
Стратегическая высота. Уверенность лидера рынка. Панорамное мышление.

**Визуальные ассоциации**
- Цвет: почти-чёрный `#080F1A`, чистый белый, акцент indigo `#5B6AF0` → violet `#8B5CF6`
- Графика: изометрические блоки данных, вертикальные bar-charts как skyline, глубина перспективы
- Типографика: ultra-light + bold контраст, крупные числа как элемент дизайна
- Анимации: parallax при скролле, числа «вырастают» снизу вверх
- Плотность: высокий контраст, низкая шумность — много воздуха вокруг ключевых данных

**Слоган**

| Язык | Слоган |
|---|---|
| RU | «Экономика сверху. Решения — точнее.» |
| EN | «See further. Decide with certainty.» |
| KZ | «Жоғарыдан көр. Дәлірек шеш.» |

**Ключевые слова:** apex, altitude, panorama, strategic, above, certainty

**Risk notes**
- Indigo/violet — пересечение с Notion, Linear, некоторыми аналитическими SaaS — нужна дифференциация через форму, не только цвет
- «Apex» в слогане может звучать самонадеянно для академической аудитории
- Изометрия дорога в разработке — если UI-бюджет ограничен, концепция деградирует

## 3. Рекомендованная концепция

### ✦ Концепция A — «Signal» с элементами C

**Аргументация:**

| Критерий | A «Signal» | B «Meridian» | C «Apex View» |
|---|---|---|---|
| Уникальность в нише | Высокая | Средняя (занято) | Средняя |
| Соответствие аудитории (студент + аналитик) | Высокое | Среднее (слишком академично) | Среднее (слишком корпоративно) |
| Масштабируемость на RU/KZ/EN | Высокая | Средняя | Высокая |
| UI-реализуемость в текущем стеке | Высокая | Средняя | Требует ресурсов |
| Технологичность ощущения | Максимальная | Умеренная | Высокая |
| Risk уровень | Низкий | Средний (TM) | Средний |

**Рекомендация:**
Взять **визуальный язык A** (navy + teal, сигнальная эстетика, data-dense) + **слоган и высоту C** («See further. Decide with certainty.»). Это даёт технологичность без военного оттенка и стратегическую уверенность без корпоративного холода.

**Итоговый слоган EVision:**

| Язык | Слоган |
|---|---|
| RU | «Видеть дальше. Решать точнее.» |
| EN | «See further. Decide with certainty.» |
| KZ | «Алысты көр. Дәлірек шеш.» |

## 4. UI Feel Table — Как бренд должен ощущаться в интерфейсе

### Цвет

| Роль | Токен | Hex | Применение |
|---|---|---|---|
| Background primary | `--bg-base` | `#080F1A` | Основной фон (dark mode) |
| Background surface | `--bg-surface` | `#0F1B2D` | Карточки, панели |
| Background elevated | `--bg-elevated` | `#162235` | Модалки, выпадающие |
| Accent primary | `--accent` | `#00C9B1` | CTA, активные линии графиков, прогресс |
| Accent secondary | `--accent-2` | `#5B6AF0` | Вторичные элементы, hover |
| Text primary | `--text-primary` | `#F0F4F8` | Основной текст |
| Text secondary | `--text-secondary` | `#8A9BB0` | Подписи, лейблы |
| Text faint | `--text-faint` | `#4A5A70` | Placeholder, disabled |
| Danger | `--danger` | `#F43F5E` | Ошибки, негативные тренды |
| Success | `--success` | `#10B981` | Позитивные значения, подтверждения |
| Warning | `--warning` | `#F59E0B` | Предупреждения, неполные данные |
| Border | `--border` | `rgba(255,255,255,0.08)` | Разделители, outline |

**Light mode:** `#FFFFFF` base, `#F4F6F9` surface, акценты те же, text `#0B1628`.

---

### Типографика

| Роль | Семейство | Вес | Размер | Трекинг |
|---|---|---|---|---|
| Заголовок Hero | Inter / Manrope | 700 | 2.5–3.5rem | -0.02em |
| Заголовок секции | Inter | 600 | 1.25–1.5rem | -0.01em |
| Числовые данные (большие) | Inter Tabular / JetBrains Mono | 700 | 2–3rem | -0.02em |
| Body | Inter | 400 | 0.875rem | 0 |
| Label / Caption | Inter | 500 | 0.75rem | +0.02em (uppercase) |
| Код / индикатор-код | JetBrains Mono | 400 | 0.75rem | 0 |

**Принцип:** большие числа — главные герои. Текст обслуживает данные, не наоборот.

---

### Ритм и spacing

| Элемент | Значение |
|---|---|
| Базовая единица | 4px |
| Внутренний padding карточки | 24px (6 ед.) |
| Gap между секциями | 32–48px |
| Радиус скругления карточки | 12px |
| Радиус кнопки | 8px |
| Граница (stroke) | 1px, opacity 8–12% |

---

### Плотность информации

| Режим | Описание |
|---|---|
| **Dashboard** | Высокая плотность. 3–4 колонки. Числа крупные, подписи мелкие. |
| **Настройки / панели** | Средняя плотность. Одна колонка. Breathing room. |
| **Модалки / онбординг** | Низкая плотность. Много воздуха. Один фокус. |

---

### Анимации

| Ситуация | Тип | Длительность | Easing |
|---|---|---|---|
| Появление данных / графика | Fade + scale-up от 0.97 | 220ms | `ease-out` |
| Линии графика (Chart.js) | Draw-in слева направо | 400ms | `ease-in-out` |
| Hover на карточке | Subtle border-glow + 2px translateY | 150ms | `ease-out` |
| Открытие модалки | Fade + scale 0.95→1 | 180ms | `ease-out` |
| Переход между страницами | Cross-fade | 150ms | `linear` |
| Счётчик числа (big stat) | Count-up от предыдущего | 600ms | `ease-out` |
| Загрузка данных | Skeleton shimmer (gradient sweep) | ∞ loop | `linear` |

**Запрет:** bounce, spring, длительные transition > 500ms, parallax на мобиле.

---

### Иконки и графика

| Компонент | Стиль |
|---|---|
| Иконки | Stroke, 1.5–1.8px, 20×20px (Lucide или Phosphor) |
| Флаги стран | Круглые, 20–24px, без тени |
| Пустые состояния | Минималистичные SVG-иллюстрации, 2–3 цвета |
| Лого | Geometric, без засечек, можно с визуальной «волной» или «сигналом» |
